# Architecture Skill
Activates when a technical evaluation or architectural review is requested for the **TalentSphere** project.

## Instructions
1. Evaluate technical trade-offs between different microservices solutions (e.g., synchronous vs asynchronous communication).
2. Coordiate dependencies between the `gateway`, `ts-shared`, and peripheral services.
3. Verify that all architectural changes align with the project design documents.
4. Recommend best practices for scaling and microservices communication.

## Constraints
- Focus only on the requested architectural changes.
- Prioritize consistency across the service stack.
